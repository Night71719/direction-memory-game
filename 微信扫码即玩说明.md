# 微信扫码即玩说明

可以做到微信扫码就玩，不需要把文件夹发到手机，也不需要用户下载安装 App。

## 最推荐方案

把这个项目部署到一个公网 HTTPS 地址，例如：

```text
https://your-game.example.com
```

部署完成后：

1. 玩家 A 用微信打开游戏网址。
2. 输入昵称，创建房间。
3. 页面会显示房间二维码。
4. 玩家 B、C、D 用微信扫一扫。
5. 微信内置浏览器会直接进入游戏页，并自动填入房间号。
6. 输入昵称，点击加入，就能一起玩。

## 为什么不能直接扫 localhost

电脑本地地址：

```text
http://localhost:3000
```

只对这台电脑自己有效。手机微信扫码后，手机里的 `localhost` 指的是手机自己，不是电脑，所以打不开。

## 局域网也能扫，但不如公网稳定

如果手机和电脑在同一个 Wi-Fi，可以让电脑用局域网 IP 打开：

```text
http://电脑局域网IP:3000
```

例如：

```text
http://192.168.1.23:3000
```

然后创建房间，二维码里就是这个局域网地址，微信扫码可以进入。

但这种方式依赖同一个 Wi-Fi、防火墙、电脑不能关机，不适合发给远处的朋友。

## 真正省心的方式

```text
部署到公网 HTTPS
微信扫码打开
输入昵称
加入房间
```

## 可以部署到哪里

适合部署到：

- Render
- Railway
- Fly.io
- 腾讯云轻量服务器
- 阿里云服务器
- 微信云托管

项目里已经准备了：

- `render.yaml`
- `Procfile`
- `package.json`
- `server.js`

这些文件可以直接用于 Node.js 平台部署。

## 需要做成微信小程序吗

暂时不需要。

这个游戏用微信内置浏览器打开就能玩，成本最低、上线最快。微信小程序需要注册、配置 AppID、改成小程序框架、提交审核，适合后面要正式发布时再做。
