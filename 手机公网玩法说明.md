# 手机直接打开游戏的方式

如果想让手机不用输入电脑 IP，也不用和电脑在同一个 Wi-Fi，最省心的方式是把这个项目部署到公网。

部署完成后，你会得到一个网址，例如：

```text
https://你的游戏地址.onrender.com
```

之后玩法就是：

1. 手机直接打开这个网址。
2. 输入昵称。
3. 点击“创建房间”，这台手机就是房主。
4. 把房间号、邀请链接或房间二维码发给朋友。
5. 朋友打开同一个网址，或扫码进入，输入昵称后加入。

注意：这里的“房主”是游戏里的房主，不是服务器主机。真正的服务器放在公网平台上，所以所有手机都能连上。

## 推荐做法：部署到 Render

项目里已经加入了 `render.yaml`，适合直接部署到 Render、Railway、Fly.io、VPS 等支持 Node.js 的平台。

通用流程：

1. 把整个 `C:\game` 项目上传到 GitHub。
2. 在 Render 里新建 Web Service。
3. 选择这个 GitHub 仓库。
4. Build Command 使用：

```bash
npm install
```

5. Start Command 使用：

```bash
npm start
```

6. 部署成功后，用平台给你的网址打开游戏。

## 能不能手机自己当服务器？

理论上可以，但不推荐。

手机要真正当服务器，需要在手机上安装类似 Termux 的工具，再安装 Node.js，再运行 `server.js`，还要处理局域网、防火墙、后台保活等问题。对实际玩游戏来说，比公网部署麻烦很多。

所以推荐方案是：

```text
公网服务器负责联机
任何手机都可以打开网址
任何手机都可以创建房间成为房主
```

## 新规则

- 奇数轮：按颜色点击。屏幕上出现带颜色的箭头，玩家要按颜色找按钮。
- 偶数轮：按箭头点击。屏幕上的箭头仍然带颜色，但玩家要忽略颜色，只看方向。
- 游戏中途加入的人会先观战，并在下一轮自动加入。
